# DRACEY - Streaming Drama App

DRACEY is a premium Chinese Drama streaming platform built with **Vue 3 (Composition API)** and **Tailwind CSS**.

## Features

- **Trending & Featured Content**: Discover the latest and trending dramas.
- **VIP Access**: Exclusive content section.
- **Dub Indo**: Dedicated section for dubbed dramas.
- **Immersive Player**: Minimalist video player with auto-next functionality.
- **Responsive Design**: Optimized for mobile and desktop viewing.

## Tech Stack

- **Vue.js 3** (Composition API)
- **Tailwind CSS** (Custom theme configuration)
- **Axios** (API integration)
- **Vue Router** (Navigation)

## Getting Started

1. Check your Node.js version (v16+ recommended).
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```

## Folder Structure

- `src/components`: UI components (Navbar, Footer, DramaCard).
- `src/views`: Page views (Home, VIP, DubIndo, Detail, Player, Search).
- `src/router`: Routing configuration with Vue Router.
- `src/assets`: Static assets.
